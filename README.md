# Introduction

## Use Cases

## Unique Value Proposition (USP)

- Portable
- NixOS design helps us have a stable platform
- Privacy focused AI native design philosophy
- Unique communications features based on mesh networking
- Alternative OSs available: Android, Yocto (we don't actively support these, but you can use what you like)

We don't win on speed/performance, or on GPU, or ergonomics. We do really win on this idea that nobody has your data besides you.  We win on long term support... we are never planning to frequently EOL support in the name of trying to push people to upgrade every 2-3 years.  We would prefer you own it forever, and add additional compute based on need or want.
